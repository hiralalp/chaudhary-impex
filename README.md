# TorqBolt - Exotic Alloy Fasteners Website

A comprehensive website for exotic alloy bolting, fasteners, and forgings manufacturer.

## Project Structure

```
├── index.html              # Homepage
├── about.html              # About page
├── contact.html            # Contact page
├── products.html           # Products overview
├── specifications.html     # Technical specifications
├── certifications.html     # Company certifications
├── privacy-policy.html     # Privacy policy
├── blog.html              # Blog listing
├── sitemap.html           # HTML sitemap
├── micro-sites.html       # Micro-sites directory
├── css/
│   └── style.css          # Main stylesheet
├── products/              # Product detail pages
│   ├── stud-bolts.html
│   ├── hex-bolts.html
│   ├── socket-head-bolts.html
│   ├── nuts.html
│   ├── washers.html
│   ├── u-bolts.html
│   ├── threaded-rods.html
│   ├── screws.html
│   └── forgings.html
├── materials/             # Material-specific pages (to be created)
├── industries/            # Industry-specific pages (to be created)
├── blog/                  # Blog articles (to be created)
├── resources/             # Technical resources (to be created)
└── services/              # Service pages (to be created)
```

## Features

- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Product Categories**: Comprehensive product catalog with detailed specifications
- **Material Information**: Detailed information about various alloy materials
- **Industry Applications**: Specific solutions for different industries
- **Certifications**: ISO, PED, NACE, NORSOK, DFAR compliance information
- **Contact Forms**: Easy inquiry and quote request system

## Technologies Used

- HTML5
- CSS3 (with modern features like Grid and Flexbox)
- Responsive design principles
- Placeholder images (to be replaced with actual product images)

## Setup Instructions

1. Open `index.html` in a web browser to view the site locally
2. All pages are interconnected through navigation
3. Replace placeholder images with actual product images
4. Update contact form action to point to your backend
