Dimensional Tolerances

Metric Bolts (ISO 4759-1):

  • Width across flats: Class A = ¬±0.3mm (M6-M10), ¬±0.4mm (M12-M16)
  • Head height: ¬±0.2mm to ¬±0.5mm depending on size
  • Thread length: -0, +2P (P = thread pitch)

Imperial Bolts (ASME B18.2.1):

  • Width across flats: -0.015" to -0.045" depending on size
  • Head height: ¬±0.005" to ¬±0.015"
  • Thread length: As specified per ASME B18.2.1 Table 3
