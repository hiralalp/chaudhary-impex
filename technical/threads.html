Thread Tolerance Classes

ISO Metric Threads:

  • 6g - Fine tolerance for external threads (bolts)
  • 6H - Medium tolerance for internal threads (nuts)
  • 6g/6H - Standard combination for general purpose fasteners

Unified Threads:

  • Class 2A - External threads (bolts) - normal fit
  • Class 2B - Internal threads (nuts) - normal fit
  • Class 3A/3B - Tighter tolerance for precision applications
