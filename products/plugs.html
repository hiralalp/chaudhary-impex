Installation Guidelines

Installation Best Practices:

  • Thread Preparation: Clean threads, remove debris and old sealant
  • Sealant Application: Apply thread sealant per manufacturer instructions
  • Torque Requirements: Follow specified torque values for proper sealing
  • Inspection: Verify plug is fully seated and properly oriented
  • Pressure Testing: Test system at operating pressure before service

Common Torque Values:

  • M8: 12-15 Nm (9-11 ft-lbs)
  • M10: 20-25 Nm (15-18 ft-lbs)
  • M12: 35-42 Nm (26-31 ft-lbs)
  • M16: 70-85 Nm (52-63 ft-lbs)
  • M20: 120-145 Nm (89-107 ft-lbs)
